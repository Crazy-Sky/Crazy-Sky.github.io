题目：95. 不同的二叉搜索树 II

给定一个整数 n，生成所有由 1 … n 为节点所组成的 二叉搜索树 。

示例1:

输入: 3
输出:
[
[1,null,3,2],
[3,2,null,1],
[3,1,null,null,2],
[2,1,3],
[1,null,2,null,3]
]
解释:
以上的输出对应以下 5 种不同结构的二叉搜索树：

提示:

0 <= n <= 8

方法：分治

思路：通过分治的思想，枚举可行根节点将其他元素分为左右子树，左右子树都有可能会有不同的树结构，所以求最终的树结构时，要将左右子树不同树结构相组合，才能得出最终所有树结构。

运行数据：执行用时：2 ms，内存消耗：40.3 MB
