leetcode 2. 两数相加

题目：2. 两数相加

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例1:

输入: l1 = [2,4,3], l2 = [5,6,4]
输出: [7,0,8]
解释: 342 + 465 = 807

示例2:

输入: l1 = [0], l2 = [0]
输出: [0]

示例3:

输入: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出: [8,9,9,9,0,0,0,1]

提示:

  • 每个链表中的节点数在范围 [1, 100] 内
  • 0 <= Node.val <= 9
  • 题目数据保证列表表示的数字不含前导零

方法：模拟

思路：同时遍历两个链表，逐位计算它们的和，并与当前位置的进位值相加。此外，如果链表遍历结束后，进位值carry不为0，则需在后面增加一个节点，节点值为carry。

运行数据：执行用时：1 ms，内存消耗：41.2 MB

复杂度分析：

  • 时间复杂度：O(max(m * n))， m 和 n 分别为两个链表的长度。
  • 空间复杂度：O(1)。
